  • OverlayRenderer

    Overview

    Render Layers objects into single, usable images.

    Note: Creating renderer objects can be an expensive task. It is recommeneded that you minimize creation of renderer objects for optimum performance.

    Examples

    Rendering a Layers Object

    // Create a new render object
let renderer = OverlayRenderer()
// Render an image
renderer.composite(from: myLayers) { result in
    // Access the UIImage result here
}

    Renders myLayers to a UIImage.

    Standalone Conversion

    // Convert an image
OverlayRenderer.convert(myCiImage) { result in
    // Access the UIImage result here
}

    Converts myCiImage to a UIImage.

  • Layers

    Overview

    Overlay works using the concept of layered images. Each layer represents an individual image that can then be added atop another layer. You can think of this like layers in Photoshop, or similar image editor.

    These images can then be used by OverlayRenderer to create a composite image.

    Create a new Layers object:

        let layers = [0: image1, 1: image2]

    guard let layers = try? Layers(with: layers) else {
        return
    }

    Examples

    Inspecting Layers

    layers.count

    The number of layers.

    layers.top

    The number of the top most layer.

    Manipulating Layers

    Accessing layers:

        layers.layer(0)

    Returns a UIImage of layer 0.

    Appending layers:

        try? layers.appendLayer(myImage)

    Appends myImage to the top of the layer stack.

    Inserting layers:

        try? layers.insertLayer(myImage, at: 0)

    Inserts myImage at layer number 0.

    Removing layers:

        layers.removeLayer(2)

    Removes layer 2.

    Updating layers:

        layers.updateLayer(4, with: myNewImage)

    Replaces the contents of layer 4 with myNewImage.

    Asset Catalog

    Layers also supports the UIImage init(named:) functionality. You can use this to easily reference images from the Asset Catalog:

        // Create a layers object
    let layers = [0: "ImageAsset1", 1: "ImageAsset2"]

    guard let layers = try? Layers(named: layers) else {
        return
    }

 try? layers.updateLayer(1, named: "ImageAsset3")
// Replace layer 1 with "ImageAsset3"

    An error will be thrown if the image could not be located.
