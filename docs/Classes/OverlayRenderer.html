OverlayRenderer

public final class OverlayRenderer

Overview

Render Layers objects into single, usable images.

Note: Creating renderer objects can be an expensive task. It is recommeneded that you minimize creation of renderer objects for optimum performance.

Examples

Rendering a Layers Object

// Create a new render object
let renderer = OverlayRenderer()
// Render an image
renderer.composite(from: myLayers) { result in
    // Access the UIImage result here
}

Renders myLayers to a UIImage.

Standalone Conversion

// Convert an image
OverlayRenderer.convert(myCiImage) { result in
    // Access the UIImage result here
}

Converts myCiImage to a UIImage.
